자바에서의 각 변수 용어 특징 정리
1. 필드(field) : 객체의 속성을 작성하는 클래스 내부 영역
== 필드 변수 == 멤버 변수

  1-1) 인스턴스 변수 : 필드에 작성되는 일반 변수
       ex) private int age;
        -> new 연산자에 의해 동적메모리 Heap에 할당되는 객체 안의 변수
        -> 가비지 콜렉터에 의해 알아서 삭제됨.
	
  1-2) 클래스 변수 (== static 변수) : 필드에 static이란 예약어가 함께 작성된 변수
       ex) public static String abc;
        -> 프로그램 실행 시 정적메모리 Static에 자동 할당, 프로그램 종료 시 자동 소멸.


2. 지역변수
메서드 또는 제어문 안에서 선언된 변수로 Stack 메모리에 저장됨.
Stack에 메서드 또는 제어문이 실행 시 할당되었다가 종료 시에 소멸.
